# EineClass
EineClass is a collection of one-button macros for various classes & specs for Turtle WoW.

Author: 101arg101 (Fryn on Nordanaar)

# Prerequisites
- Install [SuperWoW](https://github.com/balakethelock/SuperWoW/releases)
- Install [SuperAPI](https://github.com/balakethelock/SuperAPI)
- Install [Cursive](https://github.com/pepopo978/Cursive)
- Install [Nampower](https://gitea.com/avitasia/nampower/releases)
- Install [UnitXP](https://github.com/jrc13245/UnitXP_SP3)
- Install [ShaguDPS](https://github.com/shagu/ShaguDPS)

# Commands
```
/eine reload           | reloads setup variables
/eine config           | shows settings menu
/eine list             | lists rotations available to your class
/eine help             | lists available commands
/eine name of rotation | executes a one-button rotation
```

# Usage
To use any supported rotation, create a macro (run the command /macro or open up the macro window from the escape menu) with the same format as below

# Example macro
<code>/eine feral bleed</code>
